# BasicGUIExperimenting
This is a basic user interface utilizing the Java swing library. It allows users to send infromation about either a monkey or a dog to a local Microsoft Access Database.

# Features
- Creates multiple UI windows, using JFrame.
- Allows user to input information such as animal height, date acquired, training phase, etc.
- Utilizes multiple Swing elemtns, such as JFrame, text boxes, drop down menus, and a custom formatted date box.
- Basic SQL queries.
- Interfaces with an Access database to store, retrieve, and update user inputs.
